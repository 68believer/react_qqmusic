# react_qqmusic

基于 React 全家桶的 qq 音乐项目实战。技术栈：基于 react + redux + react-router + styled-components + axios + ant design + react-redux + redux-thunk immutable + redux-immutable + react-transition-group 等。UI 界面参考了 PC 桌面版的音乐、flex 布局。
